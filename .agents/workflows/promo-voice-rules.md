# Promo Voice Rules

> These rules apply ONLY to promotional content (product sales posts). For educational/value content, use `/content-voice-rules` instead.

## The Only Goal

Every promo post has one job: **get someone to click the Stripe link.**

Not to educate. Not to go viral. Not to show expertise. Click → buy.

---

## Hormozi Value Equation (Apply to Every Post)

Perceived Value = (Dream Outcome × Perceived Likelihood) / (Time Delay × Effort/Sacrifice)

**To increase perceived value:**
- Maximize the dream outcome (what does their life look like after?)
- Increase likelihood (proof: "I use this myself every day")
- Shrink time delay ("5 minutes to set up", "instant download")
- Minimize effort ("no coding", "copy-paste ready", "follow the guide")

---

## The 5-Part Formula (Use Every Time)

```
1. OUTCOME HOOK     → What does the buyer's life look like after?
2. PROBLEM BRIDGE   → What frustrating thing does this replace?
3. WHAT THEY GET    → 3 bullet specifics (numbers > vague claims)
4. PRICE ANCHOR     → What would this cost elsewhere?
5. CTA              → One link. Nothing else.
```

**Example applying the formula:**

```
Never chase a late invoice again. [OUTCOME]

Most freelancers spend 3 hours/week on payment follow-up. [PROBLEM]

This kit handles it for you:
→ 7-email escalation sequence (written, ready to copy)
→ Google Sheet tracker that auto-flags overdue
→ Text templates for awkward "hey where's my money" moments [WHAT THEY GET]

A bookkeeper would charge $300+ to set this up. [ANCHOR]

$19 → [link] [CTA]
```

## Rules — Non-Negotiable

### ✅ DO

- **Lead with the outcome**, never the product name or mechanism
- **Use specific numbers**: "60 seconds", "5 agents", "7 emails", "$300 saved" beats "fast" and "lots of"
- **One CTA per post** — one link, one action, period
- **Anchor the price** — compare to a consultant, course, or the manual alternative
- **Write in second person**: "you get", "your business", "imagine if you never had to..."
- **Short sentences** — each line is a punch
- **Include proof** — "I use this daily", "built this for my own business", "used it on 3 client projects"

### ❌ DON'T

- Don't open with the product name: ~~"Introducing the AI Agent Starter Kit..."~~
- Don't use vague adjectives: ~~"powerful", "amazing", "game-changing"~~
- Don't list features: ~~"Includes 47 pages of content"~~ → "41 templates you copy-paste into your business"
- Don't use multiple CTAs in one post
- Don't apologize for the price or hedge: ~~"It's only $19..."~~ → "$19."
- Don't use AI tells: ~~"In today's fast-paced world"~~, ~~"Whether you're a beginner or expert"~~

## Timing Rules

- Best posting windows: **9–11am ET** or **7–9pm ET**
- Max 1 promo post per day
- Ideal ratio: **10 value posts for every 1 promo post**
- Always post promo during high-follower hours — check analytics before scheduling

---

## Checklist Before Posting

- [ ] Opens with outcome, not product name
- [ ] Contains at least one specific number
- [ ] Has a price anchor
- [ ] Has exactly one CTA link
- [ ] Stripe link tested (click → live checkout)
- [ ] Under 280 chars OR formatted as proper thread
- [ ] Run through humanizer before posting
